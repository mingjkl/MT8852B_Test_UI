## v1.7.0 更新说明
1. 加入全局变量 SN_LENTH 用于设置SN码长度；
2. 加入SN码输入sn_lineedit的输入长度限制；
3. 加入SN码输入长度符合要求时，sn_lineedit变绿；
4. 加入SN码输入长度不合格时，触发测试将会报错和屏蔽线打开；
5. 加入屏蔽箱触发信号接收错误，打开屏蔽箱；
